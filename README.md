# Engenharia de Software com IA Aplicada

Este repositório é dedicado ao armazenamento de todos os exercícios, projetos e exemplos desenvolvidos durante a **Pós-Graduação em Engenharia de Software com IA Aplicada**. 

O objetivo é consolidar o aprendizado em técnicas de Machine Learning, Deep Learning, processamento de dados e integração de modelos de IA em sistemas de software modernos.

## 📂 Estrutura do Repositório

Cada pasta representa um tópico, exemplo ou exercício específico do curso:

| Projeto | Descrição | Tecnologias |
| :--- | :--- | :--- |
| [exemplo-00](./exemplo-00) | Rede Neural inicial para classificação de perfis de usuários. | Node.js, TensorFlow.js |

*(Novos exercícios serão adicionados conforme o progresso do curso)*

## 🛠️ Tecnologias Gerais
- **Linguagens**: JavaScript (Node.js), Python.
- **Frameworks de IA**: TensorFlow.js, TensorFlow, PyTorch, Scikit-Learn.
- **Ferramentas**: Git, Docker, Jupyter Notebooks.

## 🚀 Como utilizar este repositório
1. Clone o repositório:
   ```bash
   git clone https://github.com/mateuspamaral/engenharia-de-software-com-ia-aplicada.git
   ```
2. Navegue até a pasta do exercício desejado:
   ```bash
   cd exemplo-00
   ```
3. Siga as instruções do `README.md` específico dentro de cada pasta.

## 🎯 Objetivos de Aprendizado
- Desenvolvimento de Redes Neurais.
- Implementação de algoritmos de Classificação e Regressão.
- Engenharia de Prompt e integração com LLMs.
- Boas práticas de MLOps.
